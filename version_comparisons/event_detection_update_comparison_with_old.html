    Identify hydrological events with hydrograph separation using local minima method.

Compare local minima detection between versions¶

In [15]:
new_events_series = hysevt.events.metrics.get_event_series(new_events_list,gauge_data)
old_events_series = hysevt.events.metrics.get_event_series(old_events_list,gauge_data)

The new routine detects the same events like the old one, but the start and end times of the detected events are not the same. It seems like the new is actually a bit better, that the event starts excactly at the local minima. For some reason the old routine seems to detect starts just before the minima.
